# The Veo Programming Language

**Veo** - это системный язык программирования, на котором приятно писать. Он быстрый, как C++, но безопасный и с современным дизайном.

**Статус:** в разработке

## Почему Veo?

* **Простой синтаксис:** Код легко читать и писать. Сложные конструкции исключены по максимуму.
* **Безопасность:** Язык сам следит за памятью. Ошибки вроде *segmentation fault* практически исключены ещё на этапе сборки.
* **Скорость:** Благодаря бэкенду LLVM, программы на Veo работают максимально быстро.
* **Умный компилятор:** Он не просто ругается на ошибки, а помогает их исправлять и подсказывает, как сделать код лучше.
* **Система сборки и пакетный менеджер:** В компилятор Veo встроен пакетный менеджер и система сборки. Это значит, что будет легко создавать проекты с внешними зависимостями и не возникнет проблем с зависимостями на других устройствах.

## Пример

```veo
import std.io;

func main() {
    let message = "Hello, Veo!";
    io.Println(message);
}
```
